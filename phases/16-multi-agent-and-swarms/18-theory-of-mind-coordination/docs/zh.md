# 心智理论与涌现协调

> Li等人（arXiv:2310.10701）表明，在合作性文本游戏中，LLM智能体展现出**涌现的高阶心智理论(Theory of Mind, ToM)**——即推理另一个智能体关于第三个智能体信念的信念——但由于上下文管理和幻觉，在长程规划中失败。Riedl（arXiv:2510.05174）测量了群体中的高阶协同，发现**仅**在ToM提示条件下才产生身份关联的分化和目标导向的互补性；低容量LLM仅表现出虚假涌现。也就是说，协调涌现是有条件依赖提示和模型的，并非自由出现。本课实现一个最小的ToM感知智能体，在有和没有ToM提示的情况下运行合作任务，并根据Riedl 2025协议测量协调差异。

**类型：** 学习 + 构建
**语言：** Python（标准库）
**前置条件：** 阶段16 · 07（心智社会与辩论），阶段16 · 17（生成式智能体）
**时间：** 约75分钟

## 问题

多智能体协调常常看起来神奇：智能体分工合作、相互预判、避免冗余。通常这种“涌现”是提示工程的产物——有人告诉智能体要“协调”。去掉提示，协调也随之消失。

Riedl 2025年的发现更为严格：在受控条件下，只有当智能体被提示推理**其他智能体的心智**(ToM)时，协调才会涌现。没有ToM提示，即使强大的模型也表现出无法通过统计检验的协调模式。这对生产有影响：团队推出的“多智能体协调”功能依赖于提示，因此脆弱。

本课将ToM视为一种特定能力（推理关于信念的信念），构建一个最小的ToM感知智能体，并测量真正的协调与提示伪装的区别。

## 概念

### ToM的含义

发展心理学：3岁儿童认为每个人的内心世界都与自己相同。5岁儿童理解他人有不同的信念。7岁儿童能推理关于信念的信念（“她以为我认为球在杯子下面”）。这分别是零阶、一阶和二阶ToM。

对于LLM智能体，ToM阶次对应：

- **零阶：** 没有他人的模型。智能体仅根据自身观察行动。
- **一阶：** 智能体拥有每个其他智能体信念的模型。“爱丽丝相信X。”
- **二阶：** 智能体建模递归信念。“爱丽丝相信鲍勃相信X。”

Li等人2023年发现，在合作游戏中，LLM智能体中涌现出一阶和二阶ToM，但在长程通信场景下会退化。

### 萨莉-安妮测试简述

1985年的错误信念测试：萨莉将弹珠放入篮子A后离开。安妮将其移至篮子B。萨莉回来后会在哪里寻找？拥有一阶ToM的孩子说篮子A（萨莉的信念与现实不同）。没有ToM的孩子说篮子B。

GPT-4时代的LLM在直接陈述时能通过萨莉-安妮式测试。但当叙述较长、场景多次变换或问题间接表述时，它们会失败。这是2026年生产环境中LLM的ToM实际状态。

### Riedl的协调测量

Riedl（arXiv:2510.05174）构建了一个群体规模测试：N个智能体，合作目标，可变提示条件。测量：

1. **身份关联的分化：** 智能体是否随时间发展出稳定的角色区分？
2. **目标导向的互补性：** 智能体的行为是否相互补充（不同子任务）而非重复？
3. **高阶协同：** 一个统计度量，衡量群体是否实现了任何子集都无法单独达到的效果。

结果：仅在ToM提示条件下，所有三个指标都产生高于基线的信号。没有ToM提示时，中等容量模型的指标接近随机。大型模型在没有显式ToM提示时表现出一些协调，但效果小于显式提示。

### 协调错觉

没有统计控制，演示中的“涌现协调”常反映：

- 内嵌协调的提示工程（系统提示说“一起工作”）。
- 观察者偏差（我们看到期望的模式）。
- 事后选择成功的运行。

那些宣传“涌现协调”但无可测量信号的生产系统应被视为营销。在声称之前先测量。

### 一个最小的ToM感知智能体

结构：

```
agent state:
  own_beliefs:    {facts the agent believes}
  other_models:   {other_agent_id -> {beliefs_the_agent_attributes_to_them}}
  actions_last_N: [history of others' actions]

observation update:
  - update own_beliefs from direct observation
  - update other_models[agent_id] from their action + prior beliefs

action selection:
  - enumerate candidate actions
  - for each, predict what each other agent will do next given their modeled beliefs
  - pick action that maximizes joint outcome under those predictions
```

`other_models`属性是ToM状态。一阶ToM只保留一层。二阶ToM添加`other_models[i][other_models_of_j]`——我认为智能体i认为智能体j相信什么。

### 为何长程场景有害

Li等人记录：上下文限制导致智能体忘记哪个信念属于谁。幻觉为其他智能体模型添加虚假信念。两者都产生“我以为他认为X”的错误，并随时间累积。

论文及2024-2026后续工作中记录的缓解措施：

- **提示中显式的ToM状态。** 结构化格式：`{agent_id: belief_list}`。强制检索以保持身份-信念绑定。
- **更短的推理链。** 每轮较少的ToM更新减少复合幻觉。
- **外部ToM存储。** 将模型维护在LLM上下文之外；每轮仅注入相关部分。

### ToM在生产中失败的场景

- **对抗性设置。** 具有良好ToM的智能体更容易被操纵（你可以建模它们对你的建模，然后利用）。
- **异构团队。** 当模型不同时，适用于一个对手的ToM模型无法泛化。
- **依赖事实的任务。** ToM是关于信念的；如果正确性依赖于事实，ToM可能成为干扰。

### 你真的可以测量的协调

团队协调是真实的而非提示包装的三个实际信号：

1. **随时间互补性。** 在多轮任务中，智能体的行动是否覆盖不相交的子任务？
2. **预期。** 智能体A在T+1轮的行动是否依赖于对智能体B在T+2轮行动的正确预测？
3. **纠正。** 当智能体A在T轮误读了B的信念时，A是否在T+2轮前纠正？

这些在记录的多智能体系统中是可测量的。它们是"协调"叙事的具体版本。

## 动手构建

`code/main.py` 实现：

- `ToMAgent` — 跟踪自身信念和每个其他智能体的信念模型。
- 一个协作任务：三个智能体必须从三个盒子中收集三个代币；每个盒子只能容纳一个代币。智能体无法通信；它们通过彼此的行动推断意图。
- 两种配置：`ToMAgent`（无心智理论）和`zeroth_order`（带一级信念模型的心智理论）。
- 在200次随机试验中测量：完成率、重复率（两个智能体瞄准同一盒子）、平均完成轮数。

运行：

```
python3 code/main.py
```

预期输出：零阶智能体重复率约35%，在10轮内完成约60%的试验。一阶心智理论智能体重复率约5%，完成率约95%。差异即可测量的协调效果。

## 使用它

`outputs/skill-tom-auditor.md` 是一种审核多智能体系统声称"涌现协调"的技能。检查提示包装、相对于对照组的统计显著性以及测量的互补性。

## 发布

协调声称检查清单：

- **对照组。** 你的系统的一个版本，没有协调提示。测量两者。
- **统计检验。** 系统与对照组之间的差异在你的指标上是否在`p < 0.05`上显著？
- **互补性度量。** 随时间推移的行动不相交性，而不仅仅是最终成功。
- **失败案例日志。** 当智能体协调失败时，心智理论状态是什么样子？
- **模型容量披露。** 如果效果在较小模型上消失，请说明。

## 练习

1. 运行`code/main.py`。确认一阶心智理论将重复率降低约7倍。当扩展到5个智能体和5个盒子时，差距是否持续？
2. 实现二阶心智理论（智能体A模拟B对C的想法）。它比一阶有所改进吗？在什么任务上？
3. 向心智理论状态注入**幻觉**：每轮随机翻转一个信念。这会如何降低一阶性能？
4. 阅读Li等人（arXiv:2310.10701）。重现"长时程退化"发现：当轮次从10增加到30时，一阶心智理论性能如何变化？
5. 阅读Riedl 2025（arXiv:2510.05174）。在你的模拟日志上实现高阶协同统计量。在没有心智理论提示条件下，效果是否出现？

## 关键术语

|  术语  |  人们的说法  |  实际含义  |
|------|----------------|------------------------|
|  心智理论 | "理解他人心智" | 建模另一个智能体信念的能力。按阶数分级（0,1,2+）。 |
|  萨莉-安妮测试 | "错误信念测试" | 1985年发展心理学；大语言模型通过简单版本，失败于复杂版本。 |
|  一阶心智理论 | "A相信X" | 建模他人关于事实的信念。 |
|  二阶心智理论 | "A相信B相信X" | 递归建模更深一层。 |
|  身份关联分化 | "随时间稳定的角色" | Riedl的指标：角色持久，而非随机。 |
|  目标导向互补性 | "不相交行动" | 智能体瞄准不同子任务，而非同一任务。 |
|  高阶协同 | "群体超越任何子集" | Riedl用于真实协调的统计度量。 |
|  协调错觉 | "看起来协调" | 提示包装的协调外观，无可测量信号。 |
