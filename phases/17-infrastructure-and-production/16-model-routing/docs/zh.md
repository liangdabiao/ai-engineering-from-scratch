# 将模型路由作为一种降低成本的基本手段

> 动态代理会评估每个请求（任务类型、令牌长度、嵌入相似度、置信度），将简单查询发送给廉价模型，将复杂查询升级到前沿模型。这也称为模型级联(Model Cascading)。生产案例研究表明，在美国/英国/欧洲的部署中，在保持同等质量的情况下成本降低20-60%；在大型SaaS平台上，路由效率提升30%可实现每年六位数的节省。2026年的背景是LLM推理价格每年下降约10倍——从2022年末到2026年，GPT-4级别的令牌价格从约$0.04/M降至约$0.04/M。其中大部分下降来自于更好的服务堆栈（阶段17·04-09），而非硬件。路由是将价格下降转化为利润而不导致产品退步的方式。其失败模式是廉价模型漂移：路由将40%的请求推给较弱的模型，推理任务质量下降3-5%，但一个季度内无人察觉。需要通过在线质量指标（而非仅离线评估集）来把关路由。

**类型：** 学习
**语言：** Python（标准库，玩具级联路由模拟器）
**前置条件：** 阶段17·01（托管LLM平台），阶段17·19（AI网关）
**时间：** 约60分钟

## 学习目标

- 解释模型级联：优先使用廉价模型并检查置信度，低置信度时升级。
- 列举四种路由信号（任务分类、提示长度、与已知困难集的嵌入相似度、首次通过的自我置信度）。
- 计算在目标路由分配和质量损失容忍度下的预期混合成本。
- 指出用于捕捉廉价模型蔓延的漂移监控指标（在线质量门控）。

## 问题

你的服务每月在GPT-5上花费8万美元。你的分析显示70%的查询很简单：“巴黎现在几点了？”“改写这个句子。”一个Haiku类模型能以3%的成本完美处理这些查询。30%需要GPT-5的推理能力——编码、数学、多步规划。

如果你将70%的请求路由到廉价模型，30%到昂贵模型，你的账单将以相同的产品质量降低约65%。这就是路由。关键在于构建代理时不会导致质量退步。

## 核心概念

### 四种路由信号

1. **任务分类**：简单/复杂/代码生成/数学/聊天。可以基于规则的分类器、小型LLM（Haiku类，$0.25/M）或与标记桶的嵌入相似度。输出：路由 = 廉价/平衡/前沿。

2. **提示长度**：超过4K令牌的提示通常需要前沿模型以保证连贯性。少于500令牌的提示通常不需要。

3. **与已知困难集的嵌入相似度**：如果查询与已知困难桶接近（余弦相似度>0.88），则直接升级到前沿模型。

4. **首次通过的自我置信度**：发送给廉价模型；如果模型的log-prob显示低置信度、或模型拒绝、或输出模棱两可的语言，则在前沿模型上重试。这会使大约10%流量的P95延迟增加，但其他90%的流量节省50%以上的成本。

### 三种模式

**预路由**（前置分类器）：增加约5-10ms延迟；总体最快。

**级联**（优先廉价，低置信度升级）：中位数延迟约1.2倍（廉价运行加验证），升级时约2倍。最佳质量底线。

**集成路由**（对样本并行运行廉价和前沿模型，奖励模型选择）：质量最高，成本最高；仅用于关键A/B测试。

### 实现

AI网关（阶段17·19）公开路由功能。LiteLLM具有带回退和成本路由的`router`配置。Portkey具有守护+路由功能。Kong AI网关具有基于插件的路由功能。OpenRouter的模型市场提供推荐API。

开源解决方案：RouteLLM（LMSYS）、Not Diamond（商业）、Prompt Mule。

### 2026年价格曲线

|  模型类别 | 2022年末 | 2026年 | 变化  |
|-------------|-----------|------|--------|
|  GPT-4级质量 | ~$20/M | ~$0.40/M | 便宜50倍  |
|  前沿（GPT-5, Claude 4） | — | ~$3-10/M | 新层级  |

大部分改进来自服务效率——阶段17·04-09的核心课程转化为提供商侧的成本下降。路由使你能在应用层捕获这些收益，而无需等待所有用户迁移到廉价层级。

### 漂移是真正的风险

你的路由将40%的请求发送给廉价模型。六个月后，任务分布发生变化（用户变得更加老练，提出更长的问d题）。路由器没有注意到，因为它的分类器是在第一季度的数据上训练的。质量悄然下降。无人大声抱怨。你发现时，已经在基准测试中输给了竞争对手。

通过在线质量指标把关路由：

- 每条路由的用户点赞/点踩。
- 每条路由上保留样本（5%）的自动LLM评估。
- 升级率：如果级联的升级比例超过30%，说明廉价模型被过度路由。
- 每条路由的拒绝率。

### 你应该记住的数字

- 2026年同等质量下的路由节省：案例研究显示20-60%。
- LLM价格下降2022-2026：每年综合约10倍。
- GPT-4级别2022 vs 2026：约$20/M降至约$0.40/M。
- 级联延迟影响：中位数约1.2倍，升级时约2倍（约占10%流量）。

## 关键术语

|  术语  |  人们的说法  |  实际含义  |
|------|----------------|------------------------|
| 模型路由  |  "成本代理"  |  根据请求动态选择模型 |
| 模型级联  |  "先便宜后升级"  |  先运行便宜模型，若低置信度则升级到前沿模型 |
| 预路由  |  "先分类"  |  前置分类器；不再重新运行 |
| 集成路由  |  "并行选择"  |  运行多个模型，由奖励模型选出最佳 |
| 升级率  |  "升级百分比"  |  级联请求中升级的占比 |
| RouteLLM  |  "LMSYS路由器"  |  开源路由器库 |
| Not Diamond  |  "商业路由器"  |  模型路由SaaS产品 |
| 漂移  |  "廉价蔓延"  |  路由器未察觉的分布偏移 |
| 在线质量控制门  |  "实时检查"  |  自动对实时流量进行LLM评判采样 |
