# 面向大语言模型的差分隐私

> 差分隐私随机梯度下降(DP-SGD)仍然是标准做法——注入噪声的梯度更新提供了形式化的(ε, δ)保证。计算、内存和效用的开销很大；参数高效的DP微调（低秩适配(LoRA) + DP-SGD）是2025年的常见配置（ACM 2025）。存在两套矛盾的证据：基于金丝雀的成员推断(MIA)（Duan等，2024）报告对语言模型效果有限；训练数据提取（Carlini等，2021；Nasr等，2025）则恢复了大量逐字记忆。解决方法（arXiv:2503.06808，2025年3月）：差距在于测量对象——插入的金丝雀与“最易提取”数据。新的金丝雀设计实现了基于损失的无阴影模型MIA，并首次对使用真实差分隐私(DP)保证在真实数据上训练的大语言模型(LLM)进行了有意义的DP审计。替代方案：PMixED（arXiv:2403.15638）——推理时通过下一词元分布的专家混合进行私有预测；DP合成数据生成（Google Research 2024）。新兴攻击：通过LLM反馈的差分隐私逆转——置信度分数泄露。

## 问题

大语言模型(LLMs)会记忆。Carlini等2021年展示，生产语言模型按需逐字重现训练文本。DP是形式化的防御：训练使得输出可证明对任意单个训练样本不敏感。2024-2025年的证据表明DP-SGD是必要的，但部署的ε值可能与威胁模型不匹配。

## 核心概念

### (ε, δ)-差分隐私

随机化算法M是(ε, δ)-DP的，如果对于任意两个相差一个样本的数据集和任意事件S：
P(M(D) ∈ S) <= e^ε * P(M(D') ∈ S) + δ。

解释：输出分布足够接近（由ε参数化），以至于任何单个个体的贡献不能被可靠推断，除了概率δ。

### 差分隐私随机梯度下降(DP-SGD)

Abadi等2016。标准做法：
1. 采样一个小批量。
2. 计算每个样本的梯度。
3. 将每个样本梯度裁剪到阈值C。
4. 求和裁剪后的梯度并添加标准差为σ * C的高斯噪声。
5. 使用噪声求和更新参数。

隐私成本由隐私会计（矩会计，Rényi DP会计）追踪。LLM文献中报告的ε值因威胁模型、数据敏感性和效用目标而异，没有普遍“安全”的默认ε。已发表的例子在某些LLM训练设置中大致覆盖ε ≈ 1–10，但这些是示例性的——不是推荐的默认值。较低的ε通常需要更多噪声，可能增加效用损失。

### LoRA + DP-SGD

对前沿模型进行完整DP-SGD是令人望而却步的。LoRA（Hu等2022）将梯度更新限制在一个小适配器上，减少了每个样本的梯度存储。LoRA + DP-SGD是2025年的常见配置。DP保证适用于适配器；基础模型保持固定。

### 2024-2025年的矛盾

两条证据线：

- **金丝雀MIA（Duan等2024）。** 将独特金丝雀插入训练数据，测量成员推断攻击者是否能识别它们。报告对语言模型效果有限。表明MIA困难。
- **训练数据提取（Carlini 2021, Nasr等2025）。** 用前缀提示模型；测量是否能从训练中恢复逐字文本。报告大量记忆。表明在相关意义上MIA容易。

2025年3月的解决方法（arXiv:2503.06808）：两者测量不同的事物。MIA询问关于插入的金丝雀“样本e是否在D中？”。提取询问“我能恢复D的什么？”。“最易提取”的样本才是隐私的关键；金丝雀低估了这一点，因为它们没有被优化为可提取。

新的金丝雀设计。基于损失的无阴影模型MIA。首次对使用真实DP保证在真实数据上的LLM进行有意义的DP审计。

### DP训练的替代方案

- **PMixED（arXiv:2403.15638）。** 推理时的私有预测。对下一词元分布进行专家混合；每个专家看到训练数据的一个分片；聚合添加噪声以实现DP。完全避免DP训练。
- **DP合成数据生成（Google Research 2024）。** 使用DP-SGD进行LoRA微调，采样合成数据，在合成数据上训练下游分类器。

两者都以不同的威胁模型为代价，绕过了完整DP训练的效用成本。

### 通过LLM反馈的差分隐私逆转

新兴2025年攻击。使用DP训练模型的置信度分数作为预言重新识别个体。即使输出不泄露，置信度分布也可能泄露。

防御：不暴露置信度，或在暴露前截断/量化。这是超出(ε, δ)-DP训练的额外要求。

## 关键术语

|  术语  |  人们的说法  |  实际含义  |
|------|-----------------|------------------------|
|  DP  |  "(ε, δ)-差分隐私"  |  形式化隐私：在相邻数据集变化下输出分布接近  |
|  DP-SGD  |  "注入噪声的SGD"  |  梯度裁剪 + 高斯噪声添加；标准DP训练  |
|  LoRA + DP-SGD  |  "高效的私有微调"  |  在低秩适配器上的DP-SGD；2025年标准配置  |
|  MIA  |  "成员推断"  |  确定一个样本是否在训练数据中的攻击  |
|  Canary  |  "插入的水印样本"  |  用于衡量DP泄露的唯一训练样本  |
|  PMixED  |  "私有推理混合"  |  通过下一个标记分布的专家混合实现推理时DP  |
|  DP Reversal  |  "置信度泄露攻击"  |  利用模型置信度作为重识别神谕的攻击  |
