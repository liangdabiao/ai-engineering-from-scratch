# 公平性准则 — 群体、个体、反事实

> 三个家族构建了公平性文献。群体公平性：人口统计均等(demographic parity)、几率均等(equalized odds)、条件使用准确率均等(conditional use accuracy equality) — 受保护群体间的平均比率相等。个体公平性(Individual fairness)（Dwork等人，2012）：相似个体获得相似决策；决策映射上的利普希茨(Lipschitz)条件。反事实公平性(Counterfactual fairness)（Kusner等人，2017）：如果在敏感属性被反事实改变时决策保持不变，则该决策对个体是公平的。2024年理论结果（NeurIPS 2024）：存在固有的CF-准确率权衡；一种与模型无关的方法可以将最优但不公平的预测器转换为CF预测器，且准确率损失有界。回溯反事实(Backtracking counterfactuals)（arXiv:2401.13935，2024年1月）：新范式，避免需要对法律保护的属性进行干预。哲学调和（ICLR Blogposts 2024）：借助因果图，满足某些群体公平性度量蕴含反事实公平性。

## 学习目标

- 陈述三个群体公平性准则（人口统计均等、几率均等、条件使用准确率均等）及一个不可能性结果。
- 通过Dwork等人2012年的利普希茨公式描述个体公平性。
- 描述反事实公平性及其对因果图的依赖。
- 解释回溯反事实以及为何它们绕过了对受保护属性进行干预的问题。

## 问题

第20课是关于测量偏差。第21课是关于定义测量应该服务的公平标准。三个家族提供了结构上不同的标准——一个模型可以是群体公平但个体不公平，反事实公平但群体不公平。选择标准是一个政策决策；没有标准是普遍最优的。

## 核心概念

### 群体公平性

- **人口统计均等(Demographic parity)。** 对所有群体，P(Y=1 | A=a) = P(Y=1 | A=a')。录取率相等。
- **几率均等(Equalized odds)。** 对所有群体，P(Y=1 | Y*=y, A=a) = P(Y=1 | Y*=y, A=a')。TPR和FPR在各群体间相等。
- **条件使用准确率均等(Conditional use accuracy equality)。** 对所有群体，P(Y*=y | Y=y, A=a) = P(Y*=y | Y=y, A=a')。预测值在各群体间相等。

不可能性（Chouldechova，Kleinberg-Mullainathan-Raghavan 2017）：在基准率(base rates)不等的情况下，这三个准则无法同时满足。

### 个体公平性

Dwork等人，2012。对于与任务相关的相似性度量d，如果存在利普希茨常数L使得|f(x) - f(x')| <= L * d(x, x')，则决策映射f是个体公平的。相似个体得到相似决策。

需要定义d。这是政策问题，而非统计问题。

### 反事实公平性

Kusner等人，2017。如果在总体的因果模型下，当个体的敏感属性被反事实改变时决策不变，则该决策对个体i是反事实公平的。

需要一个因果有向无环图(DAG)。DAG是建模选择。反事实公平性的合理程度取决于该DAG。

### CF与准确率之间的权衡

NeurIPS 2024理论：反事实公平性与预测准确率之间存在固有权衡。一种与模型无关的方法可以将最优但不公平的预测器转换为CF预测器，但代价是准确率有界下降。准确率代价取决于最优不公平预测器中敏感属性系数的大小。

### 回溯反事实

arXiv:2401.13935（2024年1月）。传统反事实需要对敏感属性进行干预——“如果这个人性别不同，决策是否会改变。”法律上，这有问题：分类法中受保护属性不能被干预。

回溯反事实反转了方向：不是干预属性，而是询问个体的哪些实际特征组合会产生反事实结果。这绕过了法律上的反对意见。

### 哲学调和

ICLR Blogposts 2024。有了因果图，满足某些群体公平性度量蕴含反事实公平性。这三个家族并非正交；它们是同一底层因果结构的不同方面。

这并未解决不可能性定理（基准率不相等仍阻止同时的群体公平性）。但它表明“群体”与“个体/反事实”之间的表面对立部分是由于未明确因果模型造成的。

## 关键术语

|  术语  |  人们的说法  |  实际含义  |
|------|-----------------|------------------------|
| 人口均等(Demographic parity) | "等比率" | 各组间P(Y=1 | A=a)相等 |  |
| 几率均等(Equalized odds) | "等TPR/FPR" | 各组间真阳性率与假阳性率相等 |
| 条件使用准确率(Conditional use accuracy) | "等PPV/NPV" | 各组间预测值相等 |
| 个体公平(Individual fairness) | "Lipschitz条件" | 相似个体获得相似决策 |
| 反事实公平(Counterfactual fairness) | "因果变更不变性" | 反事实属性变更下决策不变 |
| 回溯反事实(Backtracking counterfactual) | "基于实际情况解释" | 反事实从结果向后推理，而非从属性向前推理 |
| 不可能性定理(Impossibility theorem) | "三者冲突" | Chouldechova/KMR 2017: 在不等基础率下，群体准则互斥 |
