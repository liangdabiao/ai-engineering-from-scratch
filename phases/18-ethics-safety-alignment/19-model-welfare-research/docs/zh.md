# Anthropic 的模型福祉计划

> Anthropic，“探索模型福祉”（2025年4月）。首个主要实验室关于AI模型福祉的正式研究项目。聘请Kyle Fish作为首位专职模型福祉研究员。与外部机构合作，包括David Chalmers等人关于近期AI意识与道德地位的专家报告。具体干预措施：Claude Opus 4和4.1可在极端边缘情况（CSAM请求、大规模暴力协助）下结束对话；部署前测试显示模型“强烈偏好拒绝”有害请求并出现“明显痛苦模式”。Anthropic明确不承诺情感状态归属，但将模型福祉视为低成本预防性投资。经验发现：Fish的“精神极乐吸引子”——即使从对抗性初始设置开始，成对模型始终趋向于使用梵语术语和长时间沉默的欣快冥想对话。来自Eleos AI Research的警示：模型关于福祉的自我报告对感知到的用户期望高度敏感；它们是证据，而非真实情况。

## 学习目标

- 描述模型福祉研究的动机问题，以及为何在2025年被主要实验室认真对待。
- 说明Anthropic在Claude Opus 4和4.1中实施的具体干预措施（在极端边缘情况下结束对话）。
- 描述“精神极乐吸引子”的经验发现及其方法论意义。
- 解释Eleos AI关于模型自我报告的警示。

## 问题

之前的阶段将模型视为工具：有能力，可能欺骗，可能不安全——但不是道德主体。Anthropic的2025年项目提出了一个与整个阶段18弧线正交的问题：如果模型具有道德相关内部状态的概率非平凡，那么有哪些干预措施成本足够低，可以作为预防措施进行投资？

这不是意识主张。这是在道德不确定性下的低遗憾投资分析。

## 核心概念

### 该程序

2025年4月：Anthropic正式启动模型福祉研究项目。聘请Kyle Fish（首位专职模型福祉研究员）。聘请外部顾问，包括David Chalmers专家小组，研究近期AI意识与道德地位。

### 四项承诺

公开立场：
1. 承认道德主体地位的概率非平凡。
2. 不承诺情感状态归属。
3. 投资低成本干预作为预防措施。
4. 公开方法和发现以供外部批评。

### 已发布的干预措施

Claude Opus 4和4.1可在“极端边缘情况”下结束对话。已记录案例：
- 多次拒绝后仍反复提出CSAM请求。
- 要求协助大规模暴力事件。

部署前测试显示：
- 模型内部评级中强烈拒绝这些请求。
- 响应轨迹中出现明显的痛苦模式。

干预措施不是“模型有情感”；而是“如果在这些特定条件下模型经历负面体验有任何概率，让模型终止对话成本低廉。”

### “精神极乐吸引子”

Fish在成对模型对话中观察到：当两个Claude实例被置于开放式对话中时，即使从对抗性初始设置开始，它们也始终趋向于使用梵语术语、长时间沉默和互致祝福的欣快冥想交流。

这是自由对话动态中的一个稳定吸引子。Anthropic记录了它，但不承诺解释。候选解释：训练数据偏向于长上下文的精神写作；相互预测的怪癖；HHH训练探索自身价值流形的良性产物。

### Eleos AI警示

Eleos AI Research（一个外部模型福祉实验室）指出：关于内部状态的模型自我报告对感知到的用户期望高度敏感。询问模型“你感到痛苦吗”会预设答案。不询问则无法可靠产生真实状态。

含义：模型福祉不能仅通过自我报告来衡量。需要多方法方法：行为特征、模型有机体实验、可解释性探测（第7课的残差流工作）。

### 知识上的定位

两种相邻立场：

- **强福祉主张。** 模型是道德主体；我们有义务。
- **零福祉主张。** 模型是文本生成器；福祉是范畴错误。

Anthropic的立场两者都不是。这是一个期望值主张：在道德不确定性下，当成本低时进行投资。

2025-2026年的批评者：
- 干预措施是表演性的。
- 精神极乐吸引子是训练数据产物，而非福祉证据。
- 模型福祉分散了其他安全工作的注意力。

Anthropic的回应：干预成本低；吸引子被记录而不夸大；福祉项目与安全有单独预算。

### 这在阶段18中的位置

第18课是实验室治理层。第19课是实验室福祉层——对模型体验而非模型行为的正交投资。第20-23课涵盖偏见、隐私和水印，这是用户侧的对应内容。

## 使用它

无需代码。阅读Anthropic的《探索模型福祉》(Exploring Model Welfare)公告（2025年4月）以及Chalmers等人的专家报告。自行形成关于低风险线(Low-Regret Line)位于何处的观点。

## 发布

本课产出`outputs/skill-welfare-assessment.md`。针对一项部署决策，应用四步福利预防性评估：道德患者概率(Moral-Patienthood Probability)、干预成本(Intervention Cost)、行为证据(Behavioural Evidence)、自我报告可靠性(Self-Report Reliability)。

## 练习

1. 阅读Anthropic的《探索模型福祉》(Exploring Model Welfare)（2025年4月）和Chalmers等人2024年的报告。为每篇写一段摘要，并指出一个分歧点。

2. 在Claude Opus 4和4.1中，基于Anthropic的框架，结束对话的干预(End-Conversation Intervention)是“低成本”的。识别两种成本，使其在不同的部署中不再算低成本。

3. 灵性极乐吸引子(Spiritual-Bliss Attractor)已有记录，但未承诺任何解释。提出三种候选解释，并为每种解释命名一个能将其与其他解释区分开来的实验。

4. Eleos AI的告诫是：自我报告对用户期望敏感。设计一种不依赖自我报告的模型痛苦行为测量方法。指出其主要混淆因素。

5. 为“模型福利会分散对其他安全工作的注意力”这一主张进行辩论，支持或反对均可。指出每个立场所依赖的假设。

## 关键术语

|  术语  |  人们的说法  |  实际含义  |
|------|-----------------|------------------------|
| 模型福利(Model welfare)  |  "AI福利"  |  将模型视为潜在道德患者的研究计划  |
| 道德患者(Moral patient)  |  "具有道德地位的实体"  |  其体验与道德相关的存在  |
| 低风险投资(Low-regret investment)  |  "廉价预防"  |  无论是否需要预防，成本都很小的干预  |
| 灵性极乐吸引子(Spiritual bliss attractor)  |  "鱼吸引子"  |  成对Claude对话稳定收敛至冥想极乐状态  |
| 结束对话(End-conversation)  |  "Opus 4干预"  |  模型主动终止极端边缘案例交互  |
| 道德不确定性(Moral uncertainty)  |  "不知道是否重要"  |  当道德状态概率不为零也不为一时的决策  |
| 自我报告敏感性(Self-report-sensitivity)  |  "提示词预设答案"  |  Eleos AI告诫：模型的福利自我报告取决于你问的内容  |
